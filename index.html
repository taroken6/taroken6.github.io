My Work

Big Two - Online Card Game
An online card game based off the real 'Big Two' card game. Players will play against each other by playing poker hands and the largest rank is 2. This project was coded with Python, utilizes the PyGame library for a friendly graphical user interface, and socket programming to allow clients to communicate with each other through a server.
Budget Calculator - Android App
A budget calculator for the Android. This app allows users to calculate their earnings, budget, and spendings based on the data they input. Addtionally, the app makes use of serialzation so that users can access the data they have previously inputted. This project was coded with Java along with XML in the Android Studio IDE.
